///////////////About program/////////////////
/////////////////////////////////////////////

This program is a simple logical math game.
You need to match fields with same value.
Foreach matched you get points with value
on matched field.

/////////////////////////////////////////////
/////////////////Technology//////////////////
/////////////////////////////////////////////

Program was written in C# .NET framework on 
Visual Studio 2017 Community
